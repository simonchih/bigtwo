Chinese readme only:

按滑鼠左鍵選牌，如果沒選牌按滑鼠右鍵，就Pass。
如果有選牌按滑鼠右鍵，當合法時，就是出牌。

玩法是大老二台式玩法，要注意 順子的大小比較，
是採中國玩法，比順子中最大的牌，
也就是2, 3, 4, 5, 6大於 3, 4, 5, 6, 7 (比2跟7)
10, J, Q , K, A 大於 9, 10, J, Q, K (比A跟K)
A, 2, 3, 4, 5或2, 3, 4, 5, 6最大（以2作為關鍵牌）
