Chinese readme only:

按滑鼠左鍵選牌，如果沒選牌按滑鼠右鍵，就Pass。
如果有選牌按滑鼠右鍵，當合法時，就是出牌。

玩法是大老二台式玩法，0.3.0以後順子的比較也採台灣玩法，
台灣玩法是以2-3-4-5-6為最大，A-2-3-4-5最小

0.2.3以前，要注意 順子的大小比較，
是採中國玩法，比順子中最大的牌，
